Telugu-Tamil Translation 
Dataset Source
The dataset for Telugu and Tamil translation pairs is collected from the OPUS Project, a collection of open parallel corpus for machine translation.

Dataset URL: https://opus.nlpl.eu/results/te&ta/corpus-result-table

Project Structure
MultiCCAligned.ta-te.te: Text file containing Telugu sentences.
MultiCCAligned.ta-te.ta: Text file containing Tamil sentences.
telugu_tamil.csv: Output CSV file containing Telugu and Tamil parallel sentences.
Telugu_Tamil.ipynb: Jupyter notebook for running the data processing steps.

Steps done :
1 Data Collection
2 Aggregating our own data
3 Data cleaning
4 Tokenization
5 Stemming
